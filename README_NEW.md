# Finance Tracker Pro 💳

A comprehensive, AI-powered personal finance tracking application built for India, featuring smart expense categorization, budget management, and financial wellness insights.

## ✨ Features

### Core Functionality
- **Smart Expense Tracking**: Log expenses with automatic categorization
- **Budget Management**: Set and monitor budgets with intelligent alerts
- **Savings Goals**: Track progress towards financial objectives
- **Analytics Dashboard**: Comprehensive financial insights and trends

### AI-Powered Features
- **Receipt Scanning**: Extract expense data from receipt photos using Gemini AI
- **Smart Categorization**: Automatic expense categorization with learning
- **Financial Predictions**: ML-powered expense forecasting
- **Wellness Scoring**: Personalized financial health assessment
- **AI Chat Assistant**: Natural language queries about your finances

### Advanced Features
- **Export Reports**: Generate detailed PDF reports
- **Dark Mode**: Full dark/light theme support with accessibility options
- **Mobile Optimized**: Responsive design with swipe navigation
- **Data Backup**: Import/export functionality for data portability
- **PWA Support**: Install as a native app on any device

## 🚀 Deployment Ready

This application is **completely standalone** and ready for deployment to any hosting platform. No external dependencies on GitHub Spark or special hosting requirements.

### Quick Deploy
- **Vercel**: Connect your GitHub repository and deploy instantly
- **Netlify**: Just connect your repo and deploy
- **Any Static Host**: Build and upload the `dist` folder

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **UI Components**: Radix UI, Shadcn/ui
- **Charts**: Recharts, D3.js
- **AI Integration**: Google Gemini API
- **Build Tool**: Vite
- **Animations**: Framer Motion

## 📱 Currency & Localization

Designed specifically for Indian users:
- All amounts displayed in ₹ (Indian Rupees)
- Indian financial categories and patterns
- Local payment methods support
- Indian tax and financial planning features

## 🔧 Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
